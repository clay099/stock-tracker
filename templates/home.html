About This Project

Project Goals

The website is be designed to assist with understanding the historic stock changes. It will allow for users to select in real time current stock ticket codes and review their historic performance. It will also allow for a user to save multiple stocks to their portfolio and select notification frequency to allow for daily, weekly, or monthly update on the performance of their portfolio.

The Data

This Site connects to Finhub.io API(documentation: https://finnhub.io/docs/api, & Official Python Library https://github.com/Finnhub-Stock-API). This API provides access to real-time stock API, institutional-grade fundamental and alternative data to supercharge your investment.

Please Note: the project is connected to a free API which allows for 60 calls per minute. Should you have a large portfolio of of stocks, continually refresh your portfolio or multiple other users are searching at the same time the API key may become overloaded, please wait one minute and try again.

This project's original function is to focus on stock market investments, In the futures functionality will be extended to Foreign Exchange and Crypto Currency as allowed by the API.

Project Functionality

  • Create unique user profile including hashed password
  • Edit user profile details with further security for changing your password
  • Create a mock portfolio of stock to track movement
  • See a snap shot summary of your portfolio (starting stock price (from tracking date), current stock price, change in value as a percent and Dollar Value etc.)
  • Send an email summary of your portfolio
  • Edit & Delete stocks from your portfolio
  • Sort your portfolio by stock symbol, percentage change in value or percentage change in currency
  • Review basic company details (Website, Market Capitalization, Industry etc. )
